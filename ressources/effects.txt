//   '+pattern' = FORCE-REMOVE (empty even if a KEEP matches)

//   'pattern'  = REMOVE      (empty the visual blocks)

// Precedence: SCOPE gate, then FORCE-REMOVE > KEEP > REMOVE.

// trailing ".aoc" on any pattern is auto-stripped to match the current .ao paths.
